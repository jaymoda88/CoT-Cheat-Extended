# CoT 作弊拓展 / Cheat Extended

適用於 **Course of Temptation**（CoT）的 ModLoader 作弊拓展 Mod，透過 [JML ModLoader](https://github.com/Lyoko-Jeremie/sugarcube-2-ModLoader) 加載，無需修改遊戲本體。

## 功能

| 功能 | 說明 |
|------|------|
| **停止時間** | 凍結所有時間推進 |
| **鎖定需求** | 防止所有需求值下降 |
| **時間倍速** | 支援 0.5x / 1x / 2x / 5x |
| **繞過暴露癖限制** | 裸體自由行動 ⚠ 可能造成 Bug |
| **全滿 / 重置技能** | 一鍵操作所有技能值 |
| **時間控制** | 快速推進 / 跳至指定時間點 |
| **隨身衣櫃** | 隨時開啟衣櫃換裝 |
| **事件瀏覽器** | 依標籤篩選並直接跳轉任意事件 |

## 安裝

1. 下載 [Releases](../../releases/latest) 頁面的 `.mod.zip` 檔案
2. 在遊戲左側欄點擊 **Mod 管理**（或類似按鈕），上傳該 `.mod.zip`
3. 重新整理頁面後，左側欄會出現 **⚙ 作弊選單** 按鈕

## 使用

點擊左側欄的 **⚙ 作弊選單** 即可開啟。所有設定即時生效，存檔後保留狀態。

## 相容性

- 遊戲版本：CoT v0.7.9e（含 ModLoader v2.101.1）
- 依賴：ModLoader `^2.0.0`

## 免責聲明

本 Mod 僅供個人娛樂使用。「繞過暴露癖限制」功能會跳過遊戲原有的場景限制，可能觸發未預期的行為，請自行承擔風險。
